## ‚öôÔ∏è Installation & Setup

To run this application from the source code, you need to have the Flutter SDK installed and configured on your machine, along with an IDE like VS Code or Android Studio.

### Prerequisites

1.  [Flutter SDK](https://flutter.dev/docs/get-started/install)
2.  [VS Code](https://code.visualstudio.com/) or [Android Studio](https://developer.android.com/studio)
3.  An active mobile device or emulator/simulator.

### Using the Source Code

1.  **Clone the Repository:**
    ```bash
    git clone [https://github.com/PrinceShawtz/Real-estate-app.git](https://github.com/PrinceShawtz/Real-estate-app.git)
    cd Real-estate-app
    ```
    *(Alternatively, download the `.zip` file from the GitHub page.)*

2.  **Open the Project:**
    Open the primary project folder (likely named `project2` or similar, depending on the structure) in your IDE.

3.  **Install Dependencies:**
    Run the following command to download all necessary packages:
    ```bash
    flutter pub get
    ```

4.  **Run the App:**
    Ensure an Android device or emulator is selected, and run the main file:
    ```bash
    flutter run
    # Or, run main.dart directly from your IDE.
    ```

### Running the APK (Android Only)

If you wish to test the compiled Android package directly on an emulator:

1.  **Get the APK:** Download the `app-release.apk` file (if available in a release or artifact).
2.  **Open Device Explorer:** In Android Studio, select your device emulator and click the three dots button to find "Open in Device Explorer" (or use the equivalent feature).
3.  **Transfer APK:** Navigate to the `sdcard/Downloads` directory within the device explorer.
4.  **Place the File:** Place the `app-release.apk` file into the `sdcard/Downloads` directory.
